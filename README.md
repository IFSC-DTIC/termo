# Gerador de Termo de Consentimento (LGPD/ECA)

Este é um gerador de Termo de Consentimento para coleta de dados biométricos de menores e uso de serviços de nuvem no Instituto Federal de Santa Catarina (IFSC). A ferramenta foi desenvolvida para garantir a conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD - Lei nº 13.709/2018) e o Estatuto da Criança e do Adolescente Digital ("ECA Digital" Lei nº 15.211/2025).

## Funcionalidades

*   **Geração de Termo de Consentimento:** Permite que pais ou responsáveis legais de estudantes menores de idade gerem um termo de consentimento personalizado.
*   **Conformidade Legal:** O documento gerado aborda especificamente o consentimento para:
    *   **Plataformas Educacionais Digitais:** Uso de serviços como Google Workspace for Education e Microsoft 365 Education.
    *   **Sistemas de Biometria:** Coleta de dados biométricos (imagem facial e/ou impressão digital) para controle de acesso físico e segurança no campus.
*   **Exportação em PDF:** O termo preenchido pode ser baixado como um arquivo PDF para assinatura digital.
*   **Interface Intuitiva:** Um formulário simples guia o usuário no preenchimento dos dados necessários.
